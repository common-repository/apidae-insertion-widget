Permet d'installer facilement un [Widget Apidae](https://www.apidae-tourisme.com/nouveaux-widgets/) fourni par votre Office du Tourisme.

== Description ==

Ce plugin permet de faciliter l'intégration dans Wordpress d'un [widget javascript fourni par le réseau Apidae](https://www.apidae-tourisme.com/nouveaux-widgets/).

**Attention : pour pouvoir utiliser ce plugin, vous aurez besoin de l'identifiant Widget Apidae que vous aura transmis votre Office du Tourisme.**

Ce plugin créé un "code court" a intégrer dans vos pages pour vous permettre d'ajouter un widget Apidae sans entrer la moindre ligne de code et sans modifier votre thème.

= Utilisation =

Récupérez le numéro de votre Widget Apidae auprès de votre Office du Tourisme.

Dans l'une de vos pages ou dans l'un de vos articles Wordpress, ajoutez un bloc de type "[/] Code Court". (screenshot n°1)

Dans ce code court, saisissez ``[wapidae id="123456"]`` où 123456 correspond à l'identifiant de votre Widget Apidae. (screenshot n°2)

Votre Widget est prêt, il vous suffit de publier votre page ou votre article. (screenshot n°3)

== Installation ==

= Prérequis Minimum =

* WordPress 4.7 ou plus

* PHP version 7.0 ou plus

= Installation Automatique =

L'installation automatique est l'option la plus simple. Pour faire une installation automatique de Wapidae, connectez vous à votre tableau de bord WordPress, naviguez jusqu'au menu Plugins et cliquez sur Ajouter Nouveau.

Dans la barre de recherche tapez Wapidae et cliquez sur Rechercher Plugins. Une fois que vous avez trouvé votre plugin, installez le en cliquant simplement sur Installer.

= Installation Manuelle =

La méthode d'installation manuelle implique de télécharger Wapidae et de l'upload dans votre serveur web via votre application FTP favorite. Le codex WordPress vous en dira plus <a href="https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation">ici</a>.

= Mise à jour =

La mise à jour automatique devrait fonctionner comme d'habitude. Nous vous suggérons de toujours sauvegarder votre site web avant de lancer n'importe quelle mise à jour automatisée pour éviter les problèmes imprévus.
